# less(Leaner Style Sheets)基础

它是一门CSS扩展语言, 它在CSS的语法基础上引入了变量, Mixin(混入), 
运算以及函数等功能, 大大简化了CSS的编写, 并且降低了CSS的维护成本.

- less使用

    - less变量

            @变量名: 值;

    - less编译

        使用easy less插件

    - less嵌套

        1. 可以将子元素的样式直接写在父元素里面

        eg:
            
            div{
                ---;
                p{
                    ---;
                }
            }

        2. 伪类/伪元素前要加&

        eg:

            a{
                ---;
                &:hover{
                    ---;
                }
            }

    - less运算

        1. 任何数字, 颜色或者变量都可以参加运算. less提供了+ - * / 四种运算, ()可以改变运算优先级
        2. 运算符左右两边要加空格
        3. 两个数参与运算, 如果只有一个单位, 则最后的结果就以这个单位为准
        4. 若两个数单位不一致, 最后的结果以第一个单位为准
