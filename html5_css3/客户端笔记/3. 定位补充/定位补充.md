# 定位补充

## 粘性定位

定义: 可以被认为是相对定位和固定定位的混合

特点:

1. 以浏览器的可视窗口为参照点移动元素
2. 粘性定位占有原先的位置
3. 必须添加top\left\right\bottom其中一个才有效

## 元素的显示与隐藏

- display

    display: none;      隐藏(**不再占据原来的位置**)

    display: block;     显示

- visibility

    inherit: 继承上一个父元素的可见性

    visible: 对象可视

    hidden: 对象隐藏(**继续占据原来的位置**)

    collapse: 主要用来隐藏表格的行或列

- overflow

    visible: 默认值(显示)

    auto: 此为body和textarea的默认值(根据条件添加滚动条)

    hidden: 不显示超出内容(定位时慎用)

    scroll: 滚动条
