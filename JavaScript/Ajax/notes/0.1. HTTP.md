# HTTP

## 介绍

HTTP(hypertext transfer protocol) [超文本传输协议], 协议规定了浏览器和万维网服务器之间互相通信的规则。

## 请求报文

请求报文由请求行、请求头部、空行和请求数据四部分组成。

请求行：

- 请求方法：GET、POST、PUT、DELETE等
- 请求URI：请求的资源路径
- HTTP版本：HTTP/1.1、HTTP/2.0等

请求头部：

- Host：请求的域名
- User-Agent：浏览器类型、版本、操作系统等
- Accept：可接受的响应内容类型
- Accept-Language：可接受的语言
- Accept-Encoding：可接受的编码方式
- Connection：是否保持连接
- Cookie：会话标识
- Content-Type：请求数据类型
- Content-Length：请求数据长度

空行：

请求头部与请求数据之间有一个空行。

请求数据：

请求的数据可以是表单数据、JSON数据、XML数据等。


## 响应报文

响应报文由响应行、响应头部、空行和响应数据四部分组成。

响应行：

- HTTP版本：HTTP/1.1、HTTP/2.0等
- 状态码：200、404、500等
- 状态描述：状态码的文本描述

响应头部：

- Content-Type：响应数据类型
- Content-Length：响应数据长度
- Set-Cookie：设置会话标识

空行：

响应头部与响应数据之间有一个空行。

响应数据：

响应的数据可以是HTML、JSON、XML等格式。
