# 正则: 命名捕获分组

```javascript

let str = "<a href='http://www.baidu.com'>百度</a>"

const reg = /<a href="(?<url>.*?)">(?<text>.*?)</a>/;

const res = reg.exec(str);

console.log(res.groups.url); // http://www.baidu.com
console.log(res.groups.text); // 百度

```
