# 美化表单元素

## 新的伪类选择器

1. focus(元素时聚焦样式)

    常用样式:outline\outline-offset(框框缩进)

2. checked(特指单选和多选框选中之后的样式)

## 新的伪元素选择器

placeholder(通常用它设置placeholder属性的样式)

## 常见用法

1. 重置表单元素样式
2. 设置多行文本框(textarea)是否调整尺寸(常用样式:resize:(both:两个方向都可以调整尺寸)(none:不能调整尺寸)(horizotal:水平)(vertical:垂直))
3. 文本框边缘到内容的距离(padding或text-indent)
4. 控制单选和多选的样式
