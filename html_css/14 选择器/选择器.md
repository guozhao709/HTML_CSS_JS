# 选择器

帮助你精准的选中想要的元素

## 简单选择器

1. id选择器
2. 元素选择器
3. 类选择器
4. 通配符选择器

        *{
            color:red;
        }

5. 根据属性名和属性值选择元素

        [href]{
            color:red;
        }

        [href$="----.com"]{
            color:red;
        }

6. 伪类选择器(按link;visited;hover;active顺序书写)

    选中某些元素的某种状态

         hover
        /*选中鼠标悬停时的a元素*/
        a:hover{
            color:red;
        }

        active:激活状态，鼠标按下状态
        a:active{
            color:green;
        }

        link:超链接未访问时的状态
        visited:超链接访问访问过后的状态
        a:link{
            color:red;
        }
        a:visited{
            color:bule;
        }

7. 伪元素选择器

        before
        after
        <p> saicjasnscn <span> caccs </span> sac </p>

        span::before{
            content:"<"
        }
        span::after{
            content:">"
        }
        在caccs前后加上<>

## 选择器的组合

1. 并且
2. 后代元素——空格——(div p{}div元素中的p元素)
3. 子元素——>——(div>p{})
4. 相邻兄弟元素——+——(.class+li{})
5. 后面出现的所有兄弟元素——～——
