# 常见样式声明

1. color
    
    元素内部的文字颜色
    **预设值** ：定义好的单词
    **三原色，色值** 光学三原色（RGB）
    每个颜色可以使用0-255之间的数字表达
    RGB表示法：rgb(0,0,0)

2. background-color

    元素背景颜色

3. font-size

    元素内部文字的尺寸大小
    （1）px：像素，绝对单位
    （2）em：相对单位（相对于父元素的字体大小）
        每个元素必须有字体大小，如果没有声明，则直接使用父元素的字体大小，如果没有父元素，则使用基准字号
                user agent，UA，用户代理

4. font-weight

    文字粗细程度，可以取值数字，可以取值为预设值（normal，bold）

    strong元素：重要的，不能忽略的内容

5. font-family

    文字类型
        使用多个字体匹配不同的环境
        ————，————，————，sans-serif（非衬线字体）

6. font-style:italic

    字体样式，通常用它设置斜体

    em元素：表示强调的内容（斜体）
    i元素：默认样式：斜体字体；实际使用中，通常用它表示一个图标

7. text-decoration

    文本修饰，给文本加线  underline  none

    del元素：错误的内容
    s元素：过期的内容

8. text-indent

    首行文本缩进

9. line-height（相对）

    每行文本的高度，该值越大，每行文本的距离越大
    设置行高为容器/内容的高度，可以让单行文本垂直居中
    行高设置为**纯数字**，表示相对当前元素的字体大小

10. width 宽度（容器）

11. height 高度（容器）

12. letter-space 文字间隙

13. text-align 元素内部文字水平排列  center left等
