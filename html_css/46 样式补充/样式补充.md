# 样式补充

## display:list-item

- 设置为该属性值的盒子,本质上仍然是一个块盒,但同时该盒子会附带另一个盒子

- 元素本身生成的盒子叫做主盒子,附带 盒子称为次盒子,次盒子和主盒子水平排列(类似于ul>li)

- 涉及的css属性

1. list-style-type:设置次盒子内容的种类
2. list-style-position:设置次盒子相对于主盒子的位置
3. list-style:是一个速写属性:type,position

**清空次盒子**

list-style:none;

## 图片失效时的宽高问题

- 如果img元素的图片链接失效,img元素的特性和普通行盒一样,无法设置宽高

- 解决方法:display:block/inline-block;

## 行盒中包含行块盒或可替换元素

行盒的高度与它内部的行块盒或可替换元素的高度无关,它的高度与字体大小有关

## text-algin:justify

最后一行除外,其他行分散对齐

## direction 和 writing-mode

开始和结束是相对的,但左和右是绝对的

direction设置开始的方向,有两种rtl和ltr

writing-mode:设置文字的书写方向(设置其父元素),eg:vertical-rl(从右到左,从上到下)等
