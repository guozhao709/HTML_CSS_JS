## 注释

注释为代码的阅读者提供帮助，注释不参与运行;

在HTML中，注释使用如下格式：

    <!-- neirong -->

快速添加注释 ctrl+/

## 元素

html

    <a href="https://www.bilibili.com">bilibili</a>

元素=起始标记 < a > + 结束标记< /a > + 元素内容 + 元素属性

属性 = 属性名 + 属性值

属性的分类：

-局部属性：某些元素特有的属性

-全局属性：所有元素通用

html

    <meta charset="UTF-8">

有些元素没有结束标记，这样的元素叫做**空元素**

空元素的两种写法：

    <meta charset="UTF-8">
    <meta charset="UTF-8"/>

## 元素的嵌套

元素不能相互嵌套

父元素、子元素、祖先元素、后代元素、兄弟元素（拥有同一个父元素）

## 标准的文档结构

    HTML：页面、HTML文档

    <!DOCTYPE html>  **文档声明：告诉浏览器，当前文档使用的HTML标准**
        <html lang="en">  **根元素，一个页面只能有一个，并且是其他元素的父元素或祖先元素；HTML5没有强制要写**
        <head>  **文档头，文档头内部的内容，不会显示到页面上**
        <meta charset="UTF-8">  **文档的元数据：附加信息**
        <meta name="viewport" content="width=device-width, initial-scale=1.0">  **适配手机端**
        <title>Bilibili</title>  **网页标题**
        </head>
        <body>  **文档体，页面上所有要参与显示的元素是他的子元素或后代元素**
        <!-- 超链接 -->
         <h1 title="the first html">
        第一个网页</h1>
        <a href="https://www.bilibili.com" title="B站">Bilibili</a>

        <img src="https://ts3.cn.mm.bing.net/th?id=OIP-C.Uxi0SafqSuc67Yhjl7ZwXQHaLb&w=201&h=310&c=8&rs=1&qlt=90&o=6&dpr=1.3&pid=3.1&rm=2" alt="">
        </body>
    </html>

lang属性：language，全局属性，表示该元素内部使用的文字是哪一种自然语言书写而成的 eg: cmn-hans

charset:指定网页内容的编码
