# 更多的选择器

## 更多的伪类选择器

1. first-child

选中第一个子元素（指定兄弟元素的第一个，前面不能有其他不同的兄弟元素，冒号前面是指定子元素）

first-of-type（子元素中同一类型的第一个元素）

2. last-child

选中最后一个子元素（其余同1）

3. nth-child

选中指定的第几个位置的子元素

必须是指定的元素，必须是同一个父元素下第几个子元素

even，关键字，等同于2n

odd，关键字，等同于2n+1

4. nth-of-type

选中指定的子元素中第几个某类型的元素

可以达到隔行变色的效果

## 更多的伪元素选择器

1. first-letter

选中元素中的文本中的第一个字母

2. first-line

选中元素中的文本中的第一行的文本

3. selection

选中被框选的文字（鼠标）
